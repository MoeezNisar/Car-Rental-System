## Car Rental System in C++:
A simple console-based Car Rental Management System built using C++.
This project is designed for beginners to understand Object-Oriented Programming (OOP) concepts like classes, objects, and modular design.

## Features:
 Add new cars to system
 Add and manage customers
 View all cars and customers
 Delete customers
 Create car bookings
 Generate rental receipt
 Simple console-based interface

## Technologies Used:
C++
Object-Oriented Programming (OOP)
Windows.h (for Sleep function)
Console-based UI

## Project Structure:
main.cpp   → Contains full source code

## How to Run:
Download or clone the repository
Open project in any C++ compiler (Code::Blocks / Dev C++ / VS Code)

## Compile the file:

g++ main.cpp -o car_rental

## Run the program:
./car_rental

## How It Works:
First, add cars in the system
Then add customer details
Create a booking by selecting car & customer
System generates a rental receipt with total cost
